Module: Main_Converter

Methods

<inner> convert(exp, bas, bn)

Función principal de conversiones.
Parameters:
Name Type Description
exp string Expresión numérica.
bas string Base de la expresión, solo acepta: 'bin', 'oct', 'dec' y 'hex'.
bn number Base numérica de la expresión.
Version:
  • 0.0.1
Throws:
Código 10004 Si el argumento 'bas' es diferente a los propuestos.
Type
ExpressionError
Returns:
  • Convierte a binario.
    Type
    bin
  • Convierte a octal.
    Type
    oct
  • Convierte a decimal.
    Type
    dex
  • Convierte a hexadecimal.
    Type
    hex
Example
//Conversión de decimal a binario
convert(10,'dec',10).bin() //return '1010'
//Conversión de binario a octal
convert('1010','bin',2).oct() //return '12'
